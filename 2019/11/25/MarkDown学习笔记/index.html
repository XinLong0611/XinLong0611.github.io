MarkDown学习笔记

-分类1 -分类2

在菜鸟教程上学习Markdown简要记录

工具推荐

Typora：https://typora.io/
Atom：https://atom.io/
两个都是很不错的Markdown编辑器，Typora使用更加的方便，配合快捷键的使用能够实现快速编辑，能够实时预览，使用教程可以去网上找，推荐一个不错的教程。
Typora使用教程：https://blog.csdn.net/WeiDelight/article/details/81011921
Atom不再多说，自行了解。

Game Start!

1.标题

标题用#，注意符号与文本之间要有空格

编辑样式：

# 一级标题
## 二级标题
### 三级标题
#### 四级标题

输出结果：

一级标题

二级标题

三级标题

四级标题

2.字体

编辑样式：

*斜体*
_斜体_
**粗体**
__粗体__
***粗斜体***
___粗斜体___

输出结果：

斜体
斜体
粗体
粗体
粗斜体
粗斜体

3.分割线

三个以上的星号（*）或减号（-）。
编辑样式：

***
* * *
******
---
- - -
-------

输出结果：

4.删除线

内容两边加两个波浪线（~~）

编辑样式：

~~删除线~~

输出结果：

删除线

5.下划线

使用HTML的<u>标签

编辑样式：

<u>下划线</u>

输出结果：

下划线

5.脚注

脚注是对文本的补充说明，鼠标移至脚注上会显示说明。

编辑样式：

脚注^望星Along

输出结果：

脚注^望星Along

6.列表

6.1无序列表

无序列表使用星号(*)、加号(+)或是减号(-)作为列表标记，注意符号与文本之间要有空格。

编辑样式：

* item1
* item2
+ item1
+ item2
- item1
- item2

输出结果：

  • item1
  • item2
  • item1
  • item2
  • item1
  • item2

6.2有序列表

有序列表使用数字加点（.）来表示。

编辑样式：

  1. item1
  2. item2

输出结果：

  1. item1
  2. item2

6.3列表嵌套

子列表前加四个空格。

编辑样式：

  1. item1
    - item1.1
    - item1.2
  2. item2
    - item2.1
    - item2.2

输出结果：

  1. item1
    • item1.1
    • item1.2
  2. item2
    • item2.1
    • item2.1

7.引用

使用符号>。

编辑样式：

>引用1
>引用2

输出样式：

引用1
引用2

嵌套用多个>

编辑样式：

> 引用1
>> 引用1.1
>>> 引用1.1.1

输出样式：

引用1

引用1.1

引用1.1.1

区块引用也可以和列表嵌套使用。

8.代码

段落中使用代码可以用`包括起来。

编辑样式：

输出函数：`print()`

输出结果：

输出函数：print()

代码区块使用3个`符号,可以指定语言,指定语言之后能够使代码高亮。
编辑样式：
``` java
int add(int a,int b){
return a + b;
}
```
输出结果：

9.链接

编辑样式：

[链接](链接地址)
或者
<链接地址>

输出结果：
百度
https://www.baidu.com/

高级链接

链接也可以用变量来代替，文档末尾附带变量地址，注意，两行之间至少空一行。
编辑样式：

[链接][链接变量名]

[链接变量名]: 链接地址

输出结果：
百度

10.图片

编辑样式：

![图片属性文本](图片地址 “可选标题（可以不要）”)

输出结果：

设置可选标题之后，鼠标触碰图片显示该段文本。
图片地址获取方法自己查找，提供一个知乎的回答：
插入图片技巧：https://www.zhihu.com/question/21065229
插入图片也可以使用变量，参考高级链接。
想要设置图片的宽和高就使用<img>标签。
编辑样式：

<img src=”https://raw.githubusercontent.com/XinLong0611/XinLong0611.github.io/master/images/MarkDown%E5%AD%A6%E4%B9%A0%E7%AC%94%E8%AE%B0/2%20-%20%E5%89%AF%E6%9C%AC.jpg" width=”100px” height=”225px”>

输出结果：

11.表格

在本博客上看不明显，可以在Typora上实时预览。
使用|来分割不同的单元格，使用-来分割表头和其他行。
编辑样式：

| 表头 | 表头 | 表头 |
| --- | --- | ---: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |

输出结果：

表头 表头 表头
单元格 单元格 单元格
单元格 单元格 单元格

表格中的对齐方式：
-:右对齐。
:-:居中对齐。
:-左对齐
编辑样式：

| 左对齐 | 居中 | 右对齐 |
| :--- | :---: | ---: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |

输出结果：

左对齐 居中 右对齐
单元格 单元格 单元格
单元格 单元格 单元格
